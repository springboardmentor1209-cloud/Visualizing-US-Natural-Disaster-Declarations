# Milestone 3: Geographical Pattern Analysis

##  Objective

The objective of this milestone is to analyze the geographical distribution of disasters across different states and identify high-risk regions using visual analytics.

---

## 📂 Files Included

* `dashboard_milestone3.pbix` → Power BI dashboard
* `screenshots/` → Visualization images

---

##  Visualizations Created

### 🔹 Disaster Distribution Across States (Map Visualization)

* Displays disaster occurrences across different states in the United States
* Helps identify regions with higher disaster frequency

### 🔹 Top 10 Most Affected States (Bar Chart)

* Shows the states with the highest number of disaster occurrences
* Enables comparison between the most impacted regions

### 🔹 Filters (Slicers)

* Year filter
* Incident type filter (Earthquake, Flood, Hurricane, Tornado, etc.)
* Allows dynamic and interactive analysis

---

## Key Insights

* Disaster occurrences are unevenly distributed across states
* Certain states experience higher disaster frequency compared to others
* Regional patterns can be observed in disaster occurrences
* Some states consistently appear in the top affected list
* Geographic visualization helps in identifying vulnerable regions

---

## Tools Used

* Power BI

---

## Outcome

This milestone provides a clear understanding of spatial patterns in disaster occurrences, helping to identify high-risk regions and supporting data-driven decision-making.
