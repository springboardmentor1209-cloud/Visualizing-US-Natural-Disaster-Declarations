# Milestone 4: Incident Type Analysis

## Objective

The objective of this milestone is to analyze different types of disasters and understand their frequency and distribution using visual analytics.

---

## 📂 Files Included

* `dashboard_final.pbix` → Power BI dashboard
* `screenshots/` → Visualization images
---

## 📊 Visualizations Created

### 🔹 Most Common Disaster Types (Bar Chart)

* Displays frequency of each disaster type
* Helps identify the most occurring disasters

### 🔹 Disaster Type Distribution (Pie Chart)

* Shows percentage contribution of each disaster type
* Provides a clear comparison of disaster proportions

### 🔹 Key Metrics (Cards)

* **Total Disasters:** 69.63K
* **Total Number of States:** 59

### 🔹 Filters (Slicers)

* Year filter
* State filter
* Enables dynamic analysis across regions and time

---

##Key Insights

* **Severe Storms** are the most common disaster type
* Followed by **Hurricanes and Floods**
* Some disaster types occur very rarely (e.g., Coastal Storm, Other)
* Disaster distribution varies across different states
* A few disaster types dominate the overall dataset

---

##Tools Used

* Power BI

---

##Outcome

This milestone provides a clear understanding of disaster categories and their distribution, helping in identifying the most impactful disaster types and supporting better decision-making.
