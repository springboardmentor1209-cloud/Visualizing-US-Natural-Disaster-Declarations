# US Natural Disaster Analysis

## Project Overview
This project analyzes US natural disaster declarations using Tableau. It focuses on temporal trends, geographical patterns, and incident type analysis.

## Features
- Yearly disaster trend analysis
- Monthly seasonality insights
- Geographical distribution (map)
- Incident type analysis
- Program support comparison (IH vs PA)

## Tools Used
- Tableau Public
- Data Visualization

## Key Insights
- Disaster frequency has increased over time
- Severe storms and hurricanes are most common
- Texas, Florida, and California are high-risk states
- Seasonal peaks occur in specific months
