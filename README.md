# 🌍 US Natural Disaster Declarations Dashboard

## 📌 Overview

This project analyzes FEMA disaster declaration data to identify trends, patterns, and distributions of natural disasters across the United States. The analysis focuses on understanding how disaster frequency varies over time, across states, and among different incident types using data analytics and visualization techniques.

---

## 🎯 Objectives

* Analyze disaster trends over time
* Identify disaster-prone states
* Understand distribution of disaster types
* Explore changes in disaster patterns
* Present insights through interactive visualizations

---

## 📂 Dataset

* Source: FEMA Disaster Declarations Dataset
* Records: 68,000+ entries
* Features include:

  * Declaration Date
  * State
  * Incident Type
  * Region
  * Program Declarations

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Matplotlib (EDA Visualization)
* Power BI (Dashboard Creation)

---

## ⚙️ Project Workflow

1. Data Acquisition
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering (Year extraction)
5. Data Visualization using Power BI
6. Insight Generation

---

## 📊 Dashboard Visualizations

The Power BI dashboard includes:

* 📈 Line Chart – Disaster declarations over time
* 📊 Bar Chart – Top disaster-prone states
* 🟦 Treemap – Distribution of disaster types
* 📊 Stacked Column Chart – Disaster types over years
* 🎗️ Ribbon Chart – Ranking of disaster types over time
* 🍩 Donut Chart – Percentage distribution of disaster types

---

## 🔍 Key Insights

* Disaster declarations have increased significantly after 2000
* Texas, Kentucky, and Florida are among the most affected states
* Severe storms, floods, and hurricanes are the most common disasters
* Disaster trends vary across years and regions
