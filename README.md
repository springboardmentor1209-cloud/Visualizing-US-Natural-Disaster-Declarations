## 📌 Overview

This project focuses on analyzing disaster declaration data and building machine learning models to derive meaningful insights and predictions. It integrates data preprocessing, visualization, and predictive modeling using Python and Power BI.

---

## 📁 Project Structure

```
├── milestone1.ipynb             # Data preprocessing & EDA
├── milestone2.ipynb             # Machine learning model building
├── milestone2.pbix              # Power BI dashboard (ML insights)
├── Dashboard_M3&M4.pbix         # Advanced dashboards
├── us_disaster_declarations.csv # Dataset
└── README.md                    # Documentation
```

## 📊 Dataset

## 🚀 How to Run the Project

### 🔧 Step 1: Clone Repository

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 📦 Step 2: Install Dependencies

```
pip install pandas numpy matplotlib seaborn scikit-learn
```

### ▶️ Step 3: Run Notebooks

* Open Jupyter Notebook / VS Code
* Run:

  * milestone1.ipynb
  * milestone2.ipynb

* Open in Power BI Desktop:
