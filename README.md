# Disaster Incident Analysis (FEMA Dataset)

## Project Overview

This project analyzes disaster incidents in the United States using FEMA disaster declaration data. The goal is to identify patterns across time, geography, and disaster types, and to understand how disaster events relate to government assistance programs.

## Tools Used

* Python (Pandas) – Data cleaning and preprocessing
* Power BI – Data visualization and dashboard creation
* VS Code – Development environment
* GitHub – Project documentation and version control

## Data Preparation

The dataset was cleaned using Python. Key preprocessing steps included:

* Handling missing values
* Formatting date columns
* Extracting year information for temporal analysis
* Preparing structured data for visualization in Power BI

## Temporal Analysis

A time-series analysis was conducted to examine disaster declarations over the years. The visualization shows that disaster occurrences fluctuate across decades, with noticeable spikes in recent years.

## Geographic Analysis

A map visualization was used to analyze disaster distribution across states. States such as Texas, California, and Florida experience a higher number of disasters due to their exposure to hurricanes, floods, wildfires, and severe storms.

## Incident Type Analysis

The distribution of disaster types reveals that storms are the most frequent disaster events, followed by floods and hurricanes. These disasters also show strong relationships with assistance program declarations.

## Assistance Program Analysis

Two federal assistance programs were analyzed:

* Individuals & Households Program
* Public Assistance Program

The analysis shows that large-scale disasters such as storms and floods often trigger both assistance programs due to their significant impact on communities and infrastructure.

## Key Insights

* Storms are the most frequent disaster type in the dataset.
* Disaster declarations have increased over time.
* Certain states consistently experience higher disaster frequency.
* Major disasters frequently result in federal assistance program declarations.
