# Visualizing US Natural Disaster Declarations

## Overview

This project provides a comprehensive analysis and visualization of US Natural Disaster Declarations. It includes a Power BI report (`report.pbix`) that enables users to explore and understand patterns, trends, and geographic distributions of natural disaster declarations across the United States.

## Project Details

- **Repository**: springboardmentor1209-cloud/Visualizing-US-Natural-Disaster-Declarations
- **License**: MIT License
- 
## Features

### 📊 Interactive Dashboards
- Visual representation of disaster declaration data across US states
- Time-series analysis of disaster trends
- Geographic mapping of disaster hotspots
- Category-based breakdown of disaster types

### 📁 Project Contents

The main deliverable is:
- **report.pbix** - Power BI report containing interactive visualizations and analysis of US natural disaster declarations

### 📈 Data Analysis Capabilities

The report provides insights into:
- Distribution of natural disasters by state
- Temporal patterns in disaster declarations
- Types of disasters and their frequency
- Impact assessment and regional analysis
- Historical trends and forecasting capabilities

### Data Dependencies
- US Natural Disaster Declaration data (FEMA data source recommended)
